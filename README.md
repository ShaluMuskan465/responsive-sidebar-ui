# Responsive Sidebar UI

This is a responsive sidebar navigation menu built using **HTML and CSS**.  
The sidebar expands on hover and shows navigation links with icons.
This project demonstrates how to create a **dashboard-style sidebar layout** for web applications.

## Features

- Responsive sidebar design
- Hover expand animation
- Clean dashboard style UI
- Google Material Icons
- User profile section at bottom

## Technologies Used

- HTML5
- CSS3
- Google Fonts
- Material Symbols Icons

## 💡 How It Works

- The sidebar width is initially **85px**
- When the user **hovers over the sidebar**, it expands to **260px**
- Menu text appears smoothly during expansion
- Icons remain visible even when collapsed

## 📚 What I Learned

- Creating responsive layouts with CSS
- Using hover effects for UI interaction
- Working with Google Fonts
- Using Material Icons
- Structuring HTML and CSS for UI components


## How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
